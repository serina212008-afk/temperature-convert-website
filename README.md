Web Development Task 3 – Temperature Converter Website Objective

The goal of this task is to create a Temperature Converter web application where users can input a temperature in Celsius or Fahrenheit and get the converted value. This task helps improve JavaScript skills, form handling, validation, and interactive web design.

Tools Used

HTML5 – Structure and layout

CSS3 – Styling and responsive design

JavaScript – Functionality for temperature conversion and input validation

VS Code / any code editor – Development environment

Browser – Testing and preview (Chrome/Edge/Firefox)

Steps Performed

Created a project folder with HTML, CSS, and JS files.

Designed the user interface with an input field, conversion button, and display area for results.

Implemented JavaScript functions to convert between Celsius and Fahrenheit.

Added input validation to ensure users enter valid numbers only.

Styled the converter using CSS for colors, spacing, and layout.

Tested the application on multiple browsers to ensure correct functionality.

Outcome

A fully functional temperature converter web application.

Users can easily convert temperatures between Celsius and Fahrenheit.

Demonstrates practical skills in interactive web development and JavaScript programming.
